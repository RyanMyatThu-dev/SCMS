# SCMS WebApp

React JavaScript web app for the SCMS API.

```sh
npm install
npm run dev
```

WebApp URL: `https://localhost:5175`

Default API base URL is `https://localhost:7072/api`, matching the API `https` launch profile.
Override it with:

```sh
VITE_API_BASE_URL=http://localhost:5140/api npm run dev
```
