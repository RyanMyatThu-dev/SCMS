# SCMS Web API Endpoints

## Appointments
- `POST   /api/Appointments`
- `GET    /api/Appointments`
- `PATCH  /api/Appointments/{id}/status`
- `POST   /api/Appointments/{id}/reschedule`
- `GET    /api/Appointments/{id}/queue-status`
- `POST   /api/Appointments/call-next`

## Auth
- `POST   /api/Auth/register`
- `POST   /api/Auth/login`
- `POST   /api/Auth/refresh`

## Dashboards
- `GET    /api/Dashboards/dashboard`
- `GET    /api/Dashboards/patient-dashboard`

## Diseases
- `GET    /api/Diseases`
- `POST   /api/Diseases`
- `PUT    /api/Diseases`
- `DELETE /api/Diseases/{id}`

## FollowUps
- `GET    /api/FollowUps`
- `POST   /api/FollowUps`
- `POST   /api/FollowUps/{id}/complete`

## LabReports
- `POST   /api/LabReports`
- `POST   /api/LabReports/{id}/result`
- `GET    /api/LabReports`
- `GET    /api/LabReports/{id}/pdf`
